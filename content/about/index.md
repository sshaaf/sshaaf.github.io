# About Me

Shaaf is a Sr. Principal Developer Advocate at [Red Hat](https://www.linkedin.com/company/red-hat/) and a seasoned architect with over 25 years of experience in open source. Shaaf specializes in the intersection of Java, Quarkus, and Kubernetes, with a current focus on Application Modernization through Generative AI. As a contributor to [Konveyor.io](https://konveyor.io/) and the ["Kai"](https://konveyor.io/components/kai/) project, he is at the forefront of using Large Language Models to automate complex legacy migrations and integrate the Model Context Protocol (MCP) into enterprise workflows. Beyond his engineering work, Shaaf is a dedicated community catalyst. He has been the lead organizer for Copenhagen Tech Talks since 2013 and serves as a Technical Editor for [InfoQ](https://www.infoq.com/profile/Shaaf-Syed/), where he curates cutting-edge content on Java and AI/ML. A frequent speaker at international summits, Shaaf is passionate about making advanced technologies like AI/ML accessible to the Java community, frequently sharing his findings on his blog at shaaf.dev. When he isn’t hacking on open-source middleware like Keycloak or Infinispan, he can be found on the cricket pitch coaching the next generation of athletes in Denmark.

# Workshops & Hands-on Training

Shaaf delivers high-impact workshops designed for senior developers and architects navigating the shift toward AI-native environments. His sessions on **Application Modernization** leverage **Konveyor AI** to demonstrate how legacy Java EE monoliths can be transformed into modern Quarkus services with LLM assistance. He also leads deep-dives into **Agentic AI** and the **Model Context Protocol (MCP)**, showing Java developers how to build autonomous agents that interact securely with enterprise data. Additionally, his **Keycloak workshops** provide a definitive guide to securing digital frontiers, covering everything from FIPS compliance to advanced identity management in multi-cloud architectures.

# Open Source Contributions & Technical Focus

Shaaf’s GitHub profile ([sshaaf](https://github.com/sshaaf)) is centered on bridging enterprise Java stability with the cutting-edge capabilities of Generative AI.

* **[keycloak-mcp-server](https://github.com/sshaaf/keycloak-mcp-server):** A significant project creating an MCP server for Keycloak. It enables LLMs to perform user management, realm configuration, and authentication flows via standardized tool-calling.
* **[llm-jakarta](https://www.google.com/search?q=https://github.com/sshaaf/llm-jakarta):** A project bridging **Jakarta EE** with LLM capabilities, providing a blueprint for enterprise Java developers to build robust, AI-powered applications.
* **[scribe-mcp-server](https://www.google.com/search?q=https://github.com/sshaaf/scribe-mcp-server):** A specialized MCP server that generates static code analysis rules in yaml for the konveyor.io project.
* **[jgraphlet](https://www.google.com/search?q=https://github.com/sshaaf/jgraphlet):** A zero-dependency Java task pipeline library. It is built for orchestrating microservice calls and asynchronous tasks with a clean, "elegant" API.
* **[waver](https://www.google.com/search?q=https://github.com/sshaaf/waver):** A specialized tool developed to automate the generation of code tutorials. similar to Pocketflow but entirely written in Java
* **[keycloak-tutorial](https://www.google.com/search?q=https://github.com/sshaaf/keycloak-tutorial):** A JavaScript-based, hands-on guide for implementing modern identity management.
* **[neovim](https://www.google.com/search?q=https://github.com/sshaaf/neovim):** A custom, Lua-based configuration specifically tuned for **professional Java development in NeoVim**, featuring advanced LSP integration and streamlined build workflows.
